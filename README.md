# Хоккейный словарь

Англо-русский словарь хоккейных терминов — для тех, кто работает с английскими материалами, статистикой и тактическими разборами.

**Живой словарь:** [hockey-platform GitHub Pages](https://github.com) ← обновится после деплоя

---

## Как устроено

- Данные (`data/terms.json`) — отдельно от интерфейса
- Нечёткий поиск по-русски и по-английски через Fuse.js
- Неподтверждённый сленг — в `data/expert_review.json`, не смешивается с фактами
- Ноль серверов, ноль фреймворков — просто HTML/CSS/JS

## Спринт 1

20 проверенных терминов: Corsi, xG, PDO, форчек, бэкчек и др.

## Запуск локально

```bash
cd web
python -m http.server 8000
# → http://localhost:8000
```

Или любой другой локальный сервер — `file://` не работает из-за fetch() для загрузки JSON.

## Добавить термин

1. Открыть `data/terms.json`
2. Добавить объект по схеме из `data/schema.md`
3. `git commit -m "data: добавлен термин X"`

Неподтверждённый термин → в `data/expert_review.json` со статусом `unverified`.
