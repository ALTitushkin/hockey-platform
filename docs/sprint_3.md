# Sprint 3 — ТЗ
## Выход к людям: бот на VPS + база до 35 + анонс в канале

> **Версия:** v0.1 · 12.06.2026
> **Статус:** в работе
> **Результат спринта:** бот работает 24/7 без домашнего ПК, в канале анонс, через 2 недели — список реальных запросов пользователей

---

## Цель спринта — одна фраза

Бот живёт на VPS, о нём знают подписчики канала, а каждый ненайденный запрос превращается в задачу на наполнение базы.

---

## Решения, принятые до спринта

- **Хостинг:** существующий британский VPS (1 CPU / 1 GB — боту хватает с запасом). Новый VPS — только когда появится второй сервис.
- **Деплой:** systemd-юнит + GitHub Actions (push в main → ssh → git pull → restart). CI/CD-пайплайн — артефакт для портфолио.
- **Admin-панель, эксперт, визуалки** — не в этом спринте. К эксперту идём со статистикой реальных запросов (спринт 4).

---

## Что входит

### Задача 1. Лог ненайденных запросов

- [ ] Запрос без результата → строка в `bot/missing_queries.log` (дата + запрос)
- [ ] Файл в .gitignore
- [ ] Это источник наполнения базы после анонса

### Задача 2. Деплой на VPS

- [ ] `deploy/hockey-bot.service` — systemd-юнит (автостарт, рестарт при падении)
- [ ] `.github/workflows/deploy.yml` — деплой по SSH при пуше в main
- [ ] Secrets в GitHub: `VPS_HOST`, `VPS_USER`, `VPS_SSH_KEY`
- [ ] Инструкция первичной настройки в `bot/README.md`
- [ ] Перевыпустить токен в BotFather (старый засветился) → новый только на VPS

### Задача 3. База до 35 терминов

- [ ] Правила: icing, offside
- [ ] Позиции: center, winger, defenseman, goaltender
- [ ] Игровые: deke, breakaway, screen, rebound, one-timer, cycle, breakout
- [ ] Роли: enforcer (тафгай)
- [ ] Синк data/ → docs/data/ (сайт)

### Задача 4. Анонс

- [ ] Черновик поста для канала «Хоккейный Овертайм»
- [ ] Публикация (вручную, Саша)
- [ ] Через ~2 недели: разбор missing_queries.log → задачи на спринт 4

---

## Definition of Done

- [ ] Бот отвечает при выключенном домашнем ПК
- [ ] push в main → бот на VPS обновился сам
- [ ] Упавший процесс перезапускается systemd
- [ ] В базе ≥ 35 терминов, тесты проходят
- [ ] Пост опубликован
- [ ] Старый токен отозван
