## Заметки по итогам

- **AdGuard блокирует бота:** фильтрация HTTPS подменяет сертификат Telegram, Python отвергает соединение (`CERTIFICATE_VERIFY_FAILED: EE certificate key too weak`). Решение: AdGuard → Настройки → Сеть → Фильтруемые приложения → отключить фильтрацию для `python.exe`.
- Запуск на Windows: `py bot.py` (не `python`).
