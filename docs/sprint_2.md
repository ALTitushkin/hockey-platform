# Sprint 2 — ТЗ
## Telegram-бот: поиск термина прямо в мессенджере

> **Версия:** v0.1 · 12.06.2026
> **Статус:** в работе
> **Продолжительность:** без дедлайна, ~1–2 сессии
> **Результат спринта:** рабочий бот, отвечающий карточкой термина на запрос по-русски и по-английски

---

## Цель спринта — одна фраза

Человек в Telegram пишет боту «xG» или «корси» → мгновенно получает карточку термина, не выходя из мессенджера.

---

## Что входит в спринт 2

### Задача 1. Бот: ядро

- [ ] `bot/bot.py` — точка входа, polling
- [ ] `bot/search.py` — поисковый модуль поверх `data/terms.json` (отдельно от телеграм-логики, переиспользуем)
- [ ] Поиск: EN, RU, аббревиатуры, нечёткое совпадение (опечатки)
- [ ] Карточка термина: EN / RU / категория / сокращения / определение
- [ ] `/start`, `/help`
- [ ] Любое текстовое сообщение = поисковый запрос
- [ ] Если не нашлось — подсказать ближайшие варианты
- [ ] Сленг из `expert_review.json` показывать с плашкой «⚠️ требует проверки»

### Задача 2. Инфраструктура

- [ ] Токен через переменную окружения `BOT_TOKEN` (или `.env`), в репо не попадает
- [ ] `bot/requirements.txt`
- [ ] `bot/README.md` — как запустить локально
- [ ] `.gitignore`: `.env`, `__pycache__`

### Задача 3. Проверка

- [ ] Юнит-тесты поиска: «xG» → Expected Goals, «корси» → Corsi, «форчек» → Forecheck
- [ ] Локальный запуск, проверка с телефона

---

## Definition of Done

- [ ] Бот запускается одной командой локально
- [ ] «xG», «корси», «PDO», «форчек» возвращают правильные карточки
- [ ] Опечатка («корс») находит термин
- [ ] Несуществующий запрос даёт внятный ответ, а не молчание
- [ ] Токен не закоммичен
- [ ] Документация обновлена (CLAUDE.md, CHANGELOG.md)

---

## Что НЕ входит (бэклог)

- Хостинг (Railway/Render) — после локального теста
- Inline-режим (@bot термин в любом чате) — кандидат на спринт 3
- Расширение базы, визуальные схемы, admin-панель

---

## Архитектурное решение

Бот читает тот же `data/terms.json`, что и сайт — принцип «одно ядро, много обёрток».
Поисковая логика в `bot/search.py` не зависит от Telegram: её можно дернуть из тестов или будущего API.
