# Спека вордмарка «КУЛЬТУРА ХОККЕЯ» — направление A

> Микро-решение по глифу «У». Версия 0.1 · 14.06.2026.
> Для дев-чата (рисует SVG). Не финальный SVG — параметры построения.
> Шрифт: Oswald (вес под макет A — SemiBold/600). Все размеры в долях **cap-height (H)**, чтобы не зависеть от кегля.

## Решение

**Вариант 1 (утверждён PM): усечь выносной хвост «У» по базовой линии, характер Oswald сохранить.**
Отвергнуты: (2) альт-глиф без дескендера — лишняя самодеятельность при том же результате; (3) другой дисплейный шрифт — уводит узнаваемость с каналом.

**Базовое правило бренда:** ни один глиф в вордмарке не свешивается ниже базовой линии. Оптический низ строки — ровный. Это и есть цель.

## Глиф «У» — построение

Берём родной контур «У» из Oswald и делаем ровно одно: **удаляем часть центрального стема ниже базовой линии, низ закрываем плоским терминалом точно на базовой линии.** Больше ничего не трогаем.

```
   cap-top  1.00H ─┐ ╲           ╱ ┌─  левый и правый диагонали — КАК В Oswald,
                   │  ╲         ╱  │   не перерисовывать, контраст сохранить
                   │   ╲       ╱   │
  fork  ~0.45H ────┼────╲─────╱────┼─  вершина-развилка — РОДНАЯ высота Oswald
                   │     │ │ │      │   (≈0.45–0.50H), НЕ поднимать/опускать
                   │     │ │ │      │   стем = ширине вертикали «К»/«Н» (w)
  baseline 0 ──────┴─────┴─┴─┴──────┴─  ← плоский срез, перпендикуляр базовой
                          ▲
                  было: хвост уходил до ≈ −0.18H — УДАЛЯЕМ
```

Параметры:
- **Низ стема:** ровно на базовой линии (0), плоский горизонтальный терминал шириной стема. Без овершута (в отличие от круглых О/С, которые у Oswald свисают на ~−0.02H — «У» теперь плоская, как К/Л/Т).
- **Высота развилки (fork):** оставить родную для Oswald (≈0.45–0.50H). Усечения хвоста достаточно — остаток стема ниже развилки ≈0.45H, «обрубком» не выглядит, ничего удлинять/опускать не нужно.
- **Толщина стема:** равна вертикали «К»/«Н» (`w ≈ 0.13H`). Диагонали — как в исходном Oswald (правый чуть тяжелее, родной контраст сохраняем).
- **Диагонали и развилку не двигаем** — отсюда «характер Oswald сохранён».

В слове «КУЛЬТУРА ХОККЕЯ» дескендер есть только у двух «У»; после правки строка садится на единую базовую линию. В «ХОККЕЯ» проблемных глифов нет.

## Кернинг вокруг «У»

Дисплейный капс трекинг по всему вордмарку: **+0.05 em (50/1000)** — для воздуха и читаемости. Поверх — локальные поджатия, компенсирующие открытую развилку «У» (под верхний-левый «карман» «У» соседи заходят):

| Пара | Поправка | Почему |
|---|---|---|
| К \| У | −0.03 em | нижний-правый луч «К» заходит под раскрытую развилку «У» |
| Т \| У | −0.03 em | свес перекладины «Т» нависает над левой ветвью «У» |
| У \| Л | −0.015 em | стыковать по нижним вертикалям, не дать «дыре» |
| У \| Р | −0.01 em | вертикаль «Р» рядом со стемом «У» — лёгкое поджатие |

Цель — чтобы «У» оптически сидела по центру своей ячейки, а не «висела» в просвете.

## Проверка Ц / Щ / Д (на будущие слова бренда)

В текущем вордмарке их нет, но правило фиксируем — чтобы сабмарки/слоганы держали ту же базовую линию:

- **Ц, Щ:** правый нижний спур **не удалять** (это опознавательный признак буквы), а **поднять так, чтобы его низ лёг на базовую линию** — короткий вынос вправо (~0.10–0.12H) на уровне базовой, без свеса вниз. Буква читается, строка не рвётся.
- **Д:** ножки/лапки — на базовую линию (Oswald обычно так и рисует; проверить и при необходимости подтянуть).
- **Я, Ф, Р, Б** и пр. — дескендеров нет, без изменений.

Разница с «У» намеренная: у «У» хвост — необязательный, убираем; у Ц/Щ спур — смысловой, сохраняем, но сажаем на базовую.

## Раскладки

**Стопкой (основная для эмблемы):**
```
КУЛЬТУРА
 ХОККЕЯ
```
- Обе строки — равной ширины. Ширину задаёт «КУЛЬТУРА» (8 знаков); «ХОККЕЯ» (6 знаков) разогнать трекингом до той же меры (флаг по обоим краям).
- Интерлиньяж (базовая→базовая) = **1.10H** (дескендеров нет → можно плотно).
- Выключка: по центру оси эмблемы.

**В строку:**
```
КУЛЬТУРА ХОККЕЯ
```
- Межсловный пробел = **0.30 em** (меньше дефолта — капс-трекинг уже раскрыл строку).
- Базовый трекинг +0.05 em + локальные кёрны выше.

| Стальной синий (акцент) | `#4A6E92` |

Назначение:
- **Стопкой:** «КУЛЬТУРА» — графит `#2A2A24`; «ХОККЕЯ» — стальной синий `#4A6E92`.
- **В строку:** «КУЛЬТУРА» графит + «ХОККЕЯ» стальной синий (то же деление), либо весь вордмарк графит с синим акцентом на «ХОККЕЯ».
- **Моно-фолбэк:** весь вордмарк `#2A2A24` (для печати/факсимиле).
- **Выворотка** (на тёмном/фото): весь вордмарк кремовый `#F1EAD9`.

Фон лок-апа — кремовый `#F1EAD9` (фон макета A), не белый.
