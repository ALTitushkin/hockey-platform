## Цель спринта

Человек утыкается в английский хоккейный термин → открывает сайт → за 3 секунды получает понятное объяснение на русском.

## Результат

Живой сайт: **https://altitushkin.github.io/hockey-platform/**

---

## Что сделано

**База терминов**
- 20 верифицированных терминов в `data/terms.json`
- Категории: статистика (11), тактика (7), правила (1), позиции (1)
- 3 сленгизма в `data/expert_review.json` — отдельно, с плашкой «требует проверки»
- Схема данных задокументирована в `data/schema.md`

**Веб-интерфейс**
- Нечёткий поиск по-русски и по-английски (Fuse.js)
- Фильтрация по категориям
- Ice-тема, мобильная вёрстка
- Сленг и неподтверждённые термины вынесены в отдельную секцию

**Инфраструктура**
- Репо: https://github.com/ALTitushkin/hockey-platform
- Хостинг: GitHub Pages, деплой автоматически при пуше в `main`
- Данные (JSON) отделены от интерфейса — можно менять UI, не трогая базу
