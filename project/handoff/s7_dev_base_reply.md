## Предложение: 2 новые категории
- **`basics`** — UI-метка «Базовое / техника». Фундамент, который не метрика / не тактика / не правило / не позиция.

(Альтернатива, если не хотим две: одна общая `meta` на хват+драфт+лиги, лиги дополнительно через cluster. Рекомендую всё же `basics`+`org` — чище для фильтров в UI.)
